# Documentacao do Projeto

Este diretorio concentra o contexto tecnico da aplicacao para manutencao e evolucao.

## Documentos

1. `docs/codex.md`
Contexto rapido do estado atual para continuidade de desenvolvimento.

2. `docs/arquitetura.md`
Visao de arquitetura, camadas, responsabilidades e fluxos principais.

3. `docs/backend.md`
Detalhes de controllers, servicos, persistencia e seguranca no backend.

4. `docs/api.md`
Referencia de endpoints REST, autorizacao e payloads.

5. `docs/banco-e-migrations.md`
Modelo de dados, seed, migrations e operacao com EF Core.

6. `docs/frontend.md`
Rotas, fluxo da interface e integracao com backend.

7. `docs/operacao-rds.md`
Como funciona a integracao com Windows/RDS e pontos de seguranca.

8. `docs/runbook.md`
Comandos praticos para rodar, buildar, publicar e validar.

9. `docs/melhorias.md`
Backlog tecnico, riscos conhecidos e trilha de evolucao.

10. `docs/dockploy.md`
Deploy da API e frontend separados em container.

11. `docs/codex-requisitos-iniciais.md`
Prompt/requisitos originais preservados.
